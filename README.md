Library Management System
​Project Overview:
This project is a comprehensive Library Management System designed to efficiently handle and organize library operations. It enables the management of book collections, categories, and tracking of data using structured SQL queries.
​Key Features:
​Database Management: Efficiently stores and manages data related to books and their categories.
​Data Analysis: Includes specialized SQL queries to calculate metrics, such as total prices per book category using advanced functions like PARTITION BY.
​Structured Organization: Built to ensure seamless retrieval and management of library resources
